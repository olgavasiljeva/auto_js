# Операторы в JavaScripte


## 1. Арифметические операторы

Арифметические операторы используются для выполнения арифметических вычислений:
- `+` — сложение
- `-` — вычитание
- `*` — умножение
- `/` — деление
- `%` — остаток от деления
- `**` — возведение в степень

Пример:
```javascript
let a = 10;
let b = 5;
let sum = a + b; // 15
```
